# 大卒になろう

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 主人公は迫りくる課題を撃ち落として、単位を取得する。
![title](<picture.png>>)
## ゲームの遊び方
* 矢印キーで主人公を操作し、スペースキーでえんぴつを発射し、課題を打ち倒す。
* BOSSを撃退後、規定単位数以上で、卒業！！
* 途中で課題にやられたらゲームオーバー。
* 得点が15点以上の時、iキーを押して1度だけ無敵状態になる
* おにぎりをとるとHPが1回復、上限は3まで
* 卒業に必要な単位数は60



## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画や操作、えんぴつの発射、単位数の表示
* 課題の出現と弾の発射
### 分担追加機能
* 規定時間が経ったら、BOSSが現れるようにする(担当:C0A24243)
Bossクラスを追加
ボスを画面上部中央に出現させる
時間経過でボスを出現させるように変更
左右にだけ移動するようにした 
ボス倒されたた時点でスコアを30獲得
再度20秒経過するとボスが出現
* 無敵機能の追加　15点以上とってiキーを押すと一回だけ10秒間無敵(担当:C0A24035)
* 残機の追加(担当:C0A24151):落ちてくる学食ランチを取得すると残機が1回復する機能


* クリア画面とゲームオーバー画面とリスタート機能の実装(担当:C0A24199)
